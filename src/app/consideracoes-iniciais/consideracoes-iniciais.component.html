Considerações iniciais

Este tutorial não pretende abordar todas as funcionalidades existentes no SEI. O foco será no uso básico/intermediário, o qual já será suficiente para a maioria dos usuários desenvolver atividades no SEI de forma bastante produtiva.


Antes de começar o Tutorial

Apesar do uso de extensões não ser imprescindível para o uso do SEI, o qual funciona de forma independente sem o uso de quaisquer delas, a fim de facilitar o uso de algumas funcionalidades, ou seja, realizar algumas tarefas de forma mais simples, prática e produtiva, é recomendada a instalação e uso de uma das seguintes extensões:

SEI++

SEI Pro

Este tutorial foi desenvolvido utilizando o SEI++, portanto algumas das funcionalidades demonstradas só poderão ser executadas de forma mais prática com o uso de uma das referidas extensões.

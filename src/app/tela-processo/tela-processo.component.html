Tipos de Documentos

No SEI existem 2 tipos de documentos:

Internos : documentos gerados/editados dentro do SEI.

Externos : documentos gerados externamente ao SEI.

Incluir Documento Interno

Para criar um documento interno clique no ícone Incluir Documento



Será exibida a tela Gerar Documento :



Selecionar o Nível de Acesso e em seguida clique no botão Confirmar Dados

Criado então o respectivo documento modelo do Órgão.

Caso a janela de edição do documento não tenha sido aberta automaticamente, clique no ícone Editar Conteúdo
